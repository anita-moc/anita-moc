## 🧠 Technical Skills

### System Administration
- Windows Server 2022 — installation, configuration, role management
- Active Directory Domain Services (AD DS) — domain setup, OUs, user/group management
- NTFS permissions — departmental folder structures (HR, IT, Finance), least privilege enforcement
- DHCP configuration — scope setup, static reservations, lease management
- DNS — forward/reverse lookup zones, AD-integrated DNS
- Group Policy (GPO) — security baselines, USB/removable media restrictions, password policies
- Proxmox virtualisation — VM deployment, network bridging, snapshot management

### Cybersecurity
- Security hardening — disabling unnecessary services, ports, and protocols on Windows Server
- GPO-based security controls — device installation restrictions, account lockout policies
- Offensive security fundamentals — Kali Linux, nmap, DNS enumeration, Metasploitable
- Network segmentation — VLAN awareness, isolated attack environments
- Access control — Role-Based Access Control (RBAC), separation of admin and standard user accounts
- VPN security — FortiClient enterprise VPN, Tailscale WireGuard tunnelling

### Scripting & Automation
